# Server Performance Stats

A simple Bash script to analyze basic Linux server performance metrics.

## Features
- Total CPU usage
- Memory usage (used vs total with percentage)
- Disk usage
- Top 5 processes by CPU usage
- Top 5 processes by memory usage
- Load average
- System uptime
- Logged-in users count

## Requirements
- Linux OS
- Bash shell
- Standard Linux utilities (`top`, `free`, `df`, `ps`, `uptime`)

## How to Run

```bash
chmod +x server-stats.sh
./server-stats.sh
